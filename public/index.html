Toy System API Documentation

Welcome to the API documentation for the Toy System project.

Authentication

Login

POST /users/login

Authenticate a user and receive a JWT token.


curl -X POST -H "Content-Type: application/json" \\
  -d '{ "email": "john.doe@example.com", "password": "password123" }' \\
  "http://localhost:3001/users/login"

Toys

Get All Toys

GET /toys

Retrieve all toys in the system.


curl -X GET "http://localhost:3001/toys"

Search Toys by Name or Info

GET /toys/search?s=query

Search toys by name or info.


curl -X GET "http://localhost:3001/toys/search?s=query"

Get Toys by Category

GET /toys/category/:catname

Retrieve toys by category.


curl -X GET "http://localhost:3001/toys/category/electronics"

Add Toy

POST /toys

Add a new toy to the system.


curl -X POST -H "Content-Type: application/json" \\
  -H "x-api-key: your_jwt_token_here" \\
  -d '{ "name": "Lego Set", "description": "Build your world with Lego bricks.", "category": "toys", "price": 29.99 }' \\
  "http://localhost:3001/toys"

Edit Toy

PUT /toys/:id

Edit an existing toy by ID.


curl -X PUT -H "Content-Type: application/json" \\
  -H "x-api-key: your_jwt_token_here" \\
  -d '{ "name": "Updated Lego Set" }' \\
  "http://localhost:3001/toys/60dcb1b8ab23a518f3b382d4"

Delete Toy

DELETE /toys/:id

Delete a toy by ID.


curl -X DELETE -H "x-api-key: your_jwt_token_here" \\
  "http://localhost:3001/toys/60dcb1b8ab23a518f3b382d4"

Get Toys by Price Range

GET /toys/prices/?min=min_price&max=max_price

Retrieve toys within a specified price range.


curl -X GET "http://localhost:3001/toys/prices/?min=10&max=50"

Get Single Toy by ID

GET /toys/single/:id

Retrieve a single toy by its ID.


curl -X GET "http://localhost:3001/toys/single/60dcb1b8ab23a518f3b382d4"

Get Count of Toys

GET /toys/count

Retrieve the total count of toys in the system.


curl -X GET "http://localhost:3001/toys/count"
